00前端儀表板（櫃台 / 員工端 Portal 升級）

把下面這整塊複製到 400前端儀表板 這個 Replit 的 Agent。
複製前把 mockup 的 screen.png 和 DESIGN.md 兩個檔案先放進專案 /design-refs/ 供 agent 讀。

# 目標
把既有 /portal/:facilityKey 員工值班入口，升級成 Stitch Dreams 的
「Kinetic Luminary」設計系統，並把五個核心區塊接到真實 API
（/api/facility-home/:groupId/home 與相關端點），零假資料。
這個 Portal 會被部署為各館電腦的瀏覽器首頁。

# 背景
- 這是 400前端儀表板 Replit，React + Vite + TS + Tailwind + shadcn/ui + Wouter。
- Admin 端已經有 8 個頁面（Dashboard/Analytics/Operations/HR Audit/
  System Health/Anomaly Reports/Announcements/Announcement Summary），
  那些頁面完全不要動。
- Portal 已經有骨架：/portal/login 用 Ragic 登入、
  /portal/:facilityKey 有 4 館 config（xinbei_pool, salu_counter,
  songshan_pool, sanmin_pool），localStorage 存 portalAuth / facilityKey。
- 後端 API（不同域）：
  * 上游 https://line-bot-assistant-ronchen2.replit.app
  * 已開 /api/facility-home/:groupId/home
        /api/facility-home/:groupId/announcements
        /api/facility-home/:groupId/announcements/:id
  * 既有 proxy 端點都走 server/routes.ts。

# 設計系統強制規範（讀 /design-refs/DESIGN.md）
- Palette：primary #001d42、primary_container #19335a、
  secondary #006b60、tertiary_fixed_dim #9dd84f、
  surface #f7f9fb、surface_container_low #f2f4f6、
  surface_container_lowest #ffffff
- Fonts：Manrope（headline/display）、Inter（body/label）
  從 Google Fonts 載入，啟用 "liga"。
- 絕對規則：
  * 禁用 1px solid border 做分隔。靠背景色階 (surface_container_low
    on surface) 表達邊界。
  * 不用 drop-shadow。用 ambient shadow：
    box-shadow: 0 30px 40px -20px rgba(25,28,30,0.06)
  * Glassmorphism：Drawer / Modal / 重點卡片用
    bg-[#19335a]/85 + backdrop-blur-xl，white text，高對比。
  * 圓角：xl (1.5rem) 或 lg (1rem)，不要銳角卡片。
  * Primary CTA：teal #006b60 背景、全圓 pill、白字。
  * Live 指標（營業中/當班中）用 lime #9dd84f。
  * 不用 pure #000 文字，用 #001d42 或 on-surface #191c1e。
- 圖示：Material Symbols Outlined（沿用 mockup）或 lucide-react 皆可，
  但同頁只能選一套。Portal 用 Material Symbols。

## 區塊資料來源（零 mock）

### (1) 櫃台交接 — col-span-8
GET /api/facility-home/:groupId/home
→ 取 response.handover[] 或 quickDocs[]（後端若只回 announcements 就先
用 announcements.filter(a => a.type === 'handover' || a.priority === 'must_read')
暫時替代，並在 TODO 註記等後端補 handover API）
- 每張小卡 pattern：icon + 標題 + 副資訊（日期 / 檔名 / 共享狀態）
- hover 顯示右側下載 / 開新頁按鈕（opacity 0 → 100）
- 空狀態：大字「尚未設定交接事項」+ 聯絡主管連結

### (2) 群組重要公告 / 規範 — col-span-4
GET /api/facility-home/:groupId/home
→ response.mustRead 包 + response.announcements.filter(type='rule')
- glass navy card (bg-[#19335a]/85, backdrop-blur-xl)
- 每條：lime pulse 點 (shadow 8px lime) + 標題 + status pill
  (OPERATIONAL=lime container, DEGRADED=amber/40)
- 底部灰字「Last check: X mins ago」（從 response.lastRefreshedAt）
- 點整張卡：開 glass drawer 顯示該群組所有 rule 類公告

### (3) 值班人員打卡 — col-span-4
GET /api/facility-home/:groupId/today-shift
（沒有此端點就先用 mock-empty state，UI 別掛掉）
列出：Payroll Portal / Facility Bookings / Equipment Ordering /
Marketing Assets — 每項右側 → icon，點擊分別導向對應系統

### (4) 公司重要公告 — col-span-8
GET /api/facility-home/:groupId/announcements?scope=facility,global&limit=3
- 3 欄人物卡（灰階頭像，hover 轉彩色）
- 頭像 / 姓名 / 職位 label（on-secondary-container teal 文字）
- 分隔用 pt-2 border-t border-outline-variant/20 （唯一允許的 hairline，
  僅 20% opacity）
- 電話 font-mono、複製按鈕
- 右上角 filter / sort 按鈕 → 開 facet drawer

### (5) 最近大活動 — col-span-4
GET /api/facility-home/:groupId/announcements?type=campaign&limit=1
&sort=startAt_asc&status=active
- navy bento with 大圖片背景 (h-48) + gradient mask from-primary via-primary/60
- lime "最近大活動" label tracking-widest
- 活動名大字 + 日期 + 地點
- LEARN MORE teal pill 按鈕
- 點擊 → /portal/:facilityKey/announcements/:id
